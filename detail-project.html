Judul project:

"Kontrol Motor Forward-Reverse dengan PLC CX-Programmer"

Deskripsi Project

Project ini mencakup pembuatan dan simulasi diagram tangga (ladder diagram) untuk mengendalikan motor agar bergerak maju (forward) dan mundur (reverse) menggunakan tombol tekan dan output PLC di software CX-Programmer. Program memungkinkan operator mengaktifkan mode forward atau reverse pada motor industri dengan interlock dan keamanan dasar melalui logika kontak dan output relay digital.
